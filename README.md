# GoIT JavaScript Homework 07 - DOM ve Etkinlik Yönetimi

Bu proje, GoIT Full Stack Developer eğitimi JavaScript modülünün yedinci ödevidir. Bu çalışmada; DOM ağacı üzerinde gezinme, dinamik öğe oluşturma, olay dinleyicileri (event listeners), form doğrulama ve kullanıcı etkileşimleri üzerine odaklanılmıştır.

## 📁 Proje Yapısı

Ödev, DOM manipülasyonu becerilerini test eden 6 farklı görevden oluşmaktadır:
